# Images

This site references high-quality food photography from Unsplash via direct URLs in the
HTML so the project works out of the box without needing to ship binary assets.

If you want to swap to **local** images instead, drop them in the matching subfolders below
and replace the corresponding `https://images.unsplash.com/...` URLs in the HTML with
`images/<subfolder>/<filename>` paths.

## Folder layout

```
images/
├── hero/        # Hero salad bowl, decorations
├── dishes/      # Menu item & special-dish photos
├── team/        # Chef / founder portraits
└── icons/       # Logo / favicon / brand marks
```
